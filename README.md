# xjs-accordion
## Simple jQuery Accordion

### HTML
**Just add this code to HTML:**

```
<div class="accordion">
        <div class="accordion__title"><a href="#">Accordion Title 1</a></div>
        <div class="accordion__panel">
            Accordion Content 1
        </div>

        <div class="accordion__title"><a href="#">Accordion Title 2</a></div>
        <div class="accordion__panel">
            Accordion Content 2
        </div>
</div>
```
